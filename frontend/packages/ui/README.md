# @lamb/ui

Shared UI components, stores, and utilities for all LAMB modules.

## Usage

```javascript
import { userStore, authService, Nav, Footer } from '@lamb/ui';
import '@lamb/ui/styles';
```

## What's included

- **Components**: Nav, Footer, LanguageSelector
- **Stores**: userStore, configStore
- **Services**: authService, configService
- **i18n**: Shared internationalization setup
- **Styles**: Tailwind theme and base utilities
