# Creator App

LAMB's main creator interface for building and managing AI-powered learning assistants.

This is part of the LAMB Frontend Monorepo. It depends on `@lamb/ui` for shared components and utilities.

## Development

```bash
cd packages/creator-app
npm run dev
```

## Building

```bash
npm run build
```

## Testing

```bash
npm run test:unit
npm run test:e2e
```
