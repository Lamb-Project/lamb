### Development Setup

See the [Installation Guide](Documentation/installationguide.md) for setting up your development environment.

### Code Style

- **Backend (Python)**: Follow PEP 8 guidelines
- **Frontend (Svelte)**: Use JavaScript with JSDoc comments (not TypeScript)
- **Documentation**: Use Markdown with clear, concise language

## ğŸŒ Translations

LAMB supports multiple languages (Basque, Catalan, Spanish, English). To add a new language:

1. Locate the translation files in `frontend/svelte-app/src/lib/i18n/`
2. Create a new locale file based on the English template
3. Translate all strings while preserving the JSON structure
4. Submit a pull request with your translation
